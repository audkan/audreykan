Bilimetrix USA

2020 - Present

Bilimetrix USA aims to provide low cost solutions to prevent kernicterus (i.e. neonatal brain damage resulting from severe jaundice) in low resource countries.

BiliPic is a smartphone Xamarin app that has the dual function of 1) providing parents and health workers easy access to training modules about jaundice in multiple languages and 2) measuring plasma bilirubin levels using color-based estimates.

SKi is a .NET MVC web app that acts as an extensive multi-language resource library for training parents and health workers. It is also the center-piece for receiving, storing, analyzing BiliPic telemetry.

I designed and developed two content management features for SKi and offline capabilities for BiliPic. I worked on increasing code coverage by 30% as well as built API services to send content and patient data between BiliPic and SKi.

KiloWatts for Humanity (KWH)

2019 - 2020

KiloWatts for Humanity is a volunteer-run non-profit organization that builds solar kiosks in Zambia and Kenya. Before each installation project, KWH must purchase various off-grid power system components from a number of suppliers. This can pose significant challenges for any volunteer-run organization, such as inventory mismanagement and internal miscommunication.

As a student software engineer, I helped design and develop an inventory management .NET web application for KWH, using C# and MySQL. I also organized and drove weekly client meetings, product demonstrations and Scrum meetings for the entirety of the project.
